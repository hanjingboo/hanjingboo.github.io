git配置

git有三个配置文件，分别是repo/.git/config,$HOME/.gitconfig,/etc/gitconfig。

git支持自动完成，也就是说你输入git conf然后按tab键，系统会自动扩展为git config。对于选项依然有效，比如git config –global –color.[tab]你会看到很多选项。
